Cosmic Erratas

A Cosmic Encounter rule compendium

Cosmic Encounter is a game where players compete for cosmic dominion, but along that journey, they must do so within the set rules of the game. And as the game grew, so did the number of often times confusing interactions. From additional expansion rules to alien specific interactions, this game has become a veritable pool of confusion.

And so this site exists. Here are compiled both the basic game rules and all of the additional information required for a smooth experience.
